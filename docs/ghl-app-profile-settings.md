## Full Description (listing / long copy)

Use this in the main app description field (adjust if there’s a character limit):

---

**Give your Voice AI agents more power with MCP.**

MCP Registry lets you discover and attach MCP (Model Context Protocol) servers from the Smithery registry directly to your HighLevel Voice AI agents. No manual config, no leaving the platform—browse, choose an agent, and attach in a few clicks.

**What you can do:**
- **Browse** — Search and explore MCP servers from the Smithery registry (name, description, tags).
- **Attach** — Select an MCP and assign it to any Voice AI agent in your location.
- **Manage** — See which MCPs are attached to each agent; change or detach anytime.

**Who it’s for:**  
Sub-account admins and agency admins who use HighLevel Voice AI and want to extend agents with tools and data from the MCP ecosystem.

**Security:**  
OAuth-based install; API keys and tokens stay on our servers. Your Custom Page uses a secure session—no secrets in the browser.

Install once per location, open the Custom Page from the app, and start connecting your Voice AI agents to MCP.
